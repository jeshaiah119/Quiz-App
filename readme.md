## 🚀 Features

🔥 Interactive MCQ-based quiz system  
⏱️ Smooth question flow (Next / Submit logic)  
📊 Real-time score tracking  
🏆 Dynamic leaderboard (stored locally)  
💾 Persistent user data using localStorage  
📱 Fully responsive UI (mobile + desktop)  
🎯 Subject-wise quiz support  
📈 Percentage-based result system  
🔐 Simple login system (no backend required)
